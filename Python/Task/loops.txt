 Beginner Level:-
--------------------------
1)Print numbers 1 to 10

2)Print even numbers between 1 and 20

3)Sum of numbers from 1 to 100

4)Print each character in a string

5)Count how many vowels are in a string

 Intermediate Level:-
-----------------------------
1)Reverse a string using a for loop

2)Find the factorial of a number

3)Print multiplication table for a given number

4)Print Fibonacci sequence up to n terms

5)Check if a number is prime

 Advanced Level:-
--------------------------
1)Nested loops: Print a number pyramid

2)Loop through a 2D list (matrix) and sum each row

3)Flatten a nested list using a loop

4)Create a histogram from a list of numbers

5)Find duplicates in a list without using set()
